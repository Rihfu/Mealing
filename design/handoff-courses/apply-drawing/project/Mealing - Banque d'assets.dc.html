Banque d'assets — figures produits
un picto par type de produit, à réutiliser partout
Jeu de figures « doodle » monoline, dans le style dessiné-main de Mealing. Pensées pour les listes de courses, le stock, les recettes et le planning. Trait currentColor (prend la couleur du rayon), remplissage translucide léger. Source réutilisable : assets/produits-assets.js
Rayons & teintes
Puces de provenance
Le picto produit dit quoi ; la puce de provenance dit pourquoi c'est sur la liste.
En contexte
Comment réutiliser
import { productSvg, findProduct }
  from './assets/produits-assets.js';

el.innerHTML = productSvg('carotte', { size: 22 });
const p = findProduct('saumon'); // → poisson
La couleur vient du parent (rayon). Un produit inconnu retombe sur le picto « panier ».
